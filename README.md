# The PokeBox

A web-based Pokémon team builder that lets you search any Pokémon and view its types and base stats. Built with vanilla HTML, CSS, and JavaScript using the PokéAPI.

---

## Features

- Search any Pokémon by name
- Displays types and all base stats instantly
- Error handling for invalid Pokémon names
- Clean, minimal UI

---

## Tech Stack

- HTML / CSS / JavaScript
- [PokéAPI](https://pokeapi.co/) — free, open Pokémon data API

---

## How to Use

1. Clone or download the repository
2. Open `index.html` in your browser
3. Type a Pokémon name in the search box and hit Search
4. View the Pokémon's types and base stats

No installs or dependencies required.

## API Reference

This project uses the [PokéAPI](https://pokeapi.co/) — a free, open RESTful API for Pokémon data.

Example endpoint used:
```
GET https://pokeapi.co/api/v2/pokemon/{name}
```

---

## Why I Built This

Existing Pokémon team builders are either too complex or have outdated UIs. The PokeBox aims to build a clean, fast alternative with competitive play in mind — starting simple and expanding over time.
